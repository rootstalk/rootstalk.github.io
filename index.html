CHAPTER I

There are three main steps that need to followed while building an app. The first step is to design the User Interface in the Designers, the second is to program the app in the Blocks editor and finally, the third step is to test the app as you build it to make sure it looks good and functions properly.

Let’s start with a classic- the HelloWorld program. This is a typical starter program that displays “Hello World” on the device. In this chapter, we will make a new variation of this program which allows the user to type in his/her name in the textbox and then the app will then give an output depending on what was entered in by the user. If the text entered was Jake, it will display ‘Hi Jake!’ along with a picture.
